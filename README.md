sinkdrummer2016
===============

A JavaScript version of the Max/MSP beast.

Installing
----------

    $ npm install
    $ ln -s /path/to/samples/ samples

Developing
----------

Build:

    $ npm run build

Watch changes to sinkdrummer.js and build bundle.js:

    $ npm run watch

Stop watching:

    $ npm run watch-stop

Running
-------

    $ npm start
